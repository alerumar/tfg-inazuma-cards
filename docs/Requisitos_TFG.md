# Requisitos funcionales

## Creación de cuenta / inicio de sesión

- **RF-01:** Como usuario, quiero poder crearme una cuenta para poder jugar al juego
- **RF-02:** Como usuario, quiero poder iniciar sesión en mi cuenta para poder acceder al juego

## Pantalla inicio

- **RF-03:** Como usuario, quiero poder acceder a la página inicial de la aplicación para poder abrir sobres o acceder a la tienda.
- **RF-04:** Como usuario, quiero poder abrir un sobre para poder conseguir cartas para mi colección.
- **RF-05:** Como usuario, quiero poder acceder a la tienda para recoger el regalo diario (puntos para abrir sobres)
- **RF-06:** Como usuario, quiero poder acceder al apartado de misiones para ver las misiones que tengo.

## Tienda

- **RF-07:** Como usuario, quiero poder reclamar mi premio diario para conseguir puntos abre sobre.
- **RF-08:** Como usuario, quiero poder canjear puntos de tienda para comprar puntos abre sobre.

## Misiones

- **RF-09:** Como usuario, quiero poder ver todas las misiones que tengo para ver mi progreso en las mismas
- **RF-10:** Como usuario, quiero poder reclamar los premios en aquellas misiones que ya estén al 100% para recibir mis recompensas

## Colección

- **RF-11:** Como usuario, quiero poder acceder a mi colección para ver todas las cartas que tengo y las que me faltan
- **RF-12:** Como usuario, quiero poder ver el detalle de una carta en específico para ver los detalles y atributos de esa carta
- **RF-13:** Como usuario, quiero poder ver el número de veces que tengo una carta en específico para ver si tengo la carta repetida

## Social

- **RF-14:** Como usuario, quiero poder ver el listado de amigos que tengo agregado y sus atributos para controlar las personas que tengo como amigos del juego
- **RF-15:** Como usuario, quiero poder ver si un amigo del juego está online u offline para saber si está activo en el juego ahora mismo
- **RF-16:** Como usuario, quiero poder ver las solicitudes de amistad recibidas para ver quién quiere ser amigo mío en el juego
- **RF-17:** Como usuario, quiero poder aceptar o rechazar las solicitudes de amistad recibidas para tener más amigos en el juego o, por el contrario, para que los usuarios no sean amigos míos
- **RF-18:** Como usuario, quiero poder ver las solicitudes de amistad enviadas para ver a quién le he solicitado ser amigo en el juego
- **RF-19:** Como usuario, quiero poder cancelar las solicitudes de amistad enviadas para evitar ser amigo en el juego con un usuario que no quiero que lo sea
- **RF-20:** Como usuario, quiero poder buscar amigos por nombre o ID para ampliar mi círculo de amigos en el juego
- **RF-21:** Como usuario, quiero poder enviar solicitud de amistad a otros usuarios para tener más amigos en el juego
- **RF-22:** Como usuario, quiero poder ver notificaciones que me lleguen para ver si me he perdido algo

## Intercambios

- **RF-23:** Como usuario, quiero poder elegir un amigo con el que poder compartir una carta para ofrecer un cambio de cartas
- **RF-24:** Como usuario que ofrece el intercambio, quiero poder elegir una carta que tenga repetida para intercambiarla con otro amigo por otra carta
- **RF-25:** Como usuario que recibe la oferta del intercambio, quiero poder aceptar el intercambio y ofrecer una carta que tenga repetida al otro usuario para proponer un intercambio de cartas
- **RF-26:** Como usuario que recibe la oferta del intercambio, quiero poder rechazar la oferta propuesta para evitar un intercambio indeseado
- **RF-27:** Como usuario que ofreció el intercambio, una vez recibida la propuesta del otro usuario, quiero poder aceptar el intercambio para enviar y recibir las respectivas cartas acordadas entre ambos
- **RF-28:** Como usuario que ofreció el intercambio, una vez recibida la propuesta del otro usuario, quiero poder rechazar el intercambio para recuperar mi carta ofrecida y evitar un intercambio indeseado

## Perfil

- **RF-29:** Como usuario, quiero poder acceder a mi perfil para ver los datos básicos, como nombre, id, nivel, etc.
- **RF-30:** Como usuario, quiero poder ver cuánto me queda de experiencia para subir de nivel para controlar mi progreso en el juego
- **RF-31:** Como usuario, quiero poder editar mi nombre, mi imagen, etc para actualizar mi perfil
- **RF-32:** Como usuario, quiero poder cambiar la contraseña de mi cuenta para aportar más seguridad a la misma
- **RF-33:** Como usuario, quiero poder cerrar sesión para aportar más seguridad a mi cuenta cuando no la uso

## Barajas

- **RF-34:** Como usuario, quiero poder crear barajas con cartas de mi colección para poder usarlas en partidas
- **RF-35:** Como usuario, quiero poder editar las barajas, tanto las cartas de su interior como el nombre de la baraja y otros atributos para mejorar mi mazo
- **RF-36:** Como usuario, quiero poder eliminar las barajas para evitar usar una baraja que no me interesa

## Partida

- **RF-37:** Como usuario, quiero poder crear una sala privada para invitar a un amigo a jugar una partida
- **RF-38:** Como usuario, quiero poder mandar invitación a un amigo para jugar una partida con el
- **RF-39:** Como usuario invitado, quiero poder aceptar la invitación metiendo el código de la sala para jugar con mi amigo
- **RF-40:** Como usuario, quiero poder elegir el mazo con el que jugar antes de que empiece la partida para jugar con el mazo deseado
- **RF-41:** Como usuario creador de la partida, una vez elegidos los mazos, quiero poder comenzar la partida para jugar con mi amigo
- **RF-42:** Como jugador, quiero poder ver todas mis cartas boca arriba y sus atributos para plantear mi jugada en la partida
- **RF-43:** Como jugador, quiero poder ver todas las cartas del rival boca abajo para controlar las cartas que le quedan por jugar
- **RF-44:** Como jugador, quiero poder ver la pila de descartes del rival, con las cartas boca arriba una vez que se hayan usado los 3 atributos de una carta para ver cuántas cartas le quedan por jugar a mi rival
- **RF-45:** Como jugador, quiero poder ver en todo momento el marcador para ver cómo va la partida
- **RF-46:** Como jugador, quiero poder seleccionar el atributo que quiera del jugador que quiera siempre que no se haya seleccionado previamente para jugar mis cartas de la manera óptima
- **RF-47:** Como jugador, quiero poder ver qué habilidad ha seleccionado mi rival, así como su valor numérico una vez se entre en combate para tener control de los atributos que tiene mi rival
- **RF-48:** Como jugador, una vez acabada la partida, quiero ver un resumen de la puntuación de la partida, así como el respectivo cartel de victoria o derrota para conocer el resultado de la partida
- **RF-49:** Como jugador, una vez acabada la partida, quiero ver las recompensas obtenidas para saber cuánto gano de puntos abre sobre, comodines de tienda y experiencia
- **RF-50:** Como jugador creador de la partida, una vez acabada, quiero tener la opción de darle a volver a jugar para volver a jugar con tu amigo si él acepta
- **RF-51:** Como jugador invitado a la partida, una vez acabada, quiero tener la opción de aceptar o rechazar la oferta de volver a jugar para decidir si quiero jugar otra partida o no
- **RF-52:** Como jugador, una vez acabada la partida, quiero poder volver a la sala para salir de la pantalla de la partida
- **RF-53:** Como creador de una sala, quiero poder eliminar la sala para dejar de jugar
- **RF-54:** Como invitado a una sala, quiero poder abandonar la sala cuando quiera para no jugar partida
- **RF-55:** Como jugador, quiero ver en todo momento cuánto tiempo queda para que se acabe mi turno, para controlar el tiempo restante que tengo para elegir atributo

# Requisitos no funcionales

- **RNF-01:** Para poder abrir un sobre de forma gratuita se deben esperar 6 horas.
- **RNF-02:** Para recoger el regalo diario, se deben esperar 24 horas. Se partirá de una hora predeterminada y se podrá recoger el premio una vez al día. Hasta el próximo día a esa hora establecida no se podrá reclamar el premio.
- **RNF-03:** En una partida, cada jugador tendrá 45 segundos por turno para elegir qué atributo echar. Si no elige carta, se echa aleatoriamente un atributo de una carta
- **RNF-04:** En una partida, si un jugador abandona y no reconecta en menos de 60 segundos, se acaba la partida y se le da por ganador al usuario que quedaba jugando
