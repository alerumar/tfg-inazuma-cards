# Inazuma Eleven TFG ⚡

Aplicación backend desarrollada con Spring Boot para la gestión y cálculo de cartas de jugadores de Inazuma Eleven.

## Tecnologías
- Java
- Spring Boot
- JPA
- HeidiSQL

## Objetivo
Diseñar un sistema de evaluación de cartas basado en atributos (ataque, defensa, control) y posición.
